# Вітальна книга

Статичний сайт-привітання у стилі книги з перегортанням сторінок, музикою, фотоальбомом і фінальним салютом.

## Як розмістити на GitHub Pages

1. Завантаж папку `birthday-book` у репозиторій.
2. У налаштуваннях репозиторію відкрий `Pages`.
3. Обери гілку з сайтом і папку, де лежить `index.html`.
4. Відкрий посилання, яке GitHub Pages покаже після публікації.

Музика запускається після натискання на кнопку відкриття книги, бо браузери не дозволяють автоматично вмикати звук без дії користувача.
